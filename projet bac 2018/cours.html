BIENVENUE DANS L'ONGLET COURS

Bienvenue à tous, dans ce cours complet traitant le langage informatique python.

Le but de ce cours est d’explorer les différentes fonctionnalités de python et de vous apprendre à les utiliser pas à pas.

Ce cours s’adresse à tous, des plus parfaits débutants aux personnes disposant déjà d’un bagage informatique

Python est un langage de base en informatique. Nous avons mis a votre disposition 7 chapitres differents concernant ce language . Chaque cours est composé d'une vidéo explicative et d'un résumé du cours .

Nous vous proposons aussi de télécharger le cours. sous forme de PDF. Sur ceux, je vous invite à cliquer ici pour débuter le Cours.
